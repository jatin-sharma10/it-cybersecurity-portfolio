IT & Cybersecurity Portfolio

This repository contains hands-on projects demonstrating foundational skills across:

IT support and troubleshooting

Cloud infrastructure and serverless architecture (AWS)

Networking fundamentals

Cybersecurity concepts, logging, and defensive analysis

The focus of this portfolio is clear documentation, structured system design, and real-world operational scenarios, rather than exploit development or unsafe experimentation.

Each project is scoped to reflect how systems are designed, secured, monitored, and maintained in professional environments.

Projects Included
1. Internal Records API (Serverless Hospital Infrastructure)

Design and implementation of a serverless internal records service using AWS managed services.
The system supports non-clinical operational records, emphasizing secure access, auditability, observability, and cost-aware infrastructure design.

Key areas covered include API Gateway, AWS Lambda, DynamoDB, IAM least-privilege access, CloudWatch logging, and operational monitoring.

2. Cybersecurity Incident Tracking Database (PostgreSQL)

Design of a relational database to log, track, and analyze cybersecurity incidents using structured schemas and analytical queries.
This project focuses on data modeling, normalization, and querying to support incident response and post-incident analysis workflows.

3. Server Infrastructure Design (Hybrid On-Prem + Cloud)

Architecture and disaster recovery planning for a mid-sized organization transitioning from legacy on-premises servers to a modern hybrid environment.
The project evaluates availability, backup strategies, failover considerations, and operational tradeoffs between on-prem and cloud systems.
